# Thalean Action Theory

## Naming lock

TAT means **Thalean Action Theory**.

This replaces an earlier relational descriptor as the broad conceptual umbrella.

Working lineage:

    DAT -> TGT -> TAT

    Dodecahedral Action Theory named the first shape.
    Thalean Graph Theory made the shape accountable.
    Thalean Action Theory names the principle that survived.

TAT studies action before objecthood: how constrained action becomes observable as relation, structure, trace, and witness.

Relation remains central, but it is no longer the headline descriptor. In this framing, relation is the grammar exposed by action; action is the primitive under test.

## Boundary

TGT remains the finite graph-theoretic spine.

TAT is broader. It names the action-level theory that TGT first made accountable.


# Thalean Action Theory

Thalean Action Theory is the umbrella research program for the Thalean body of work.

It studies how identity, meaning, action, and accountability arise through relational structure, admissible transformation, and invariants that survive changed frames.

Core maxim:

Identity is not isolated substance. Identity is relational invariance under admissible change.

Operational maxim:

A thing is what its relations make possible, and what its presence makes possible for others.

## Internal stack
